# Memoizing the state in React / Redux applications. Overview of the Reselect library

## Getting started

1. **`git clone`** https://github.com/serhii-devrel/redux-selectors.git

2. **`npm install`** or **`yarn install`**

3. **`npm start`** or **`yarn start`**
